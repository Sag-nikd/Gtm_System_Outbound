## Key Design Decisions

- **Pydantic v2 throughout**: all schemas use `.model_dump()` not `.dict()`; validation via `DealRecord(**record)` with ValidationError catch-and-skip at ingestion
- **Python 3.7.9 compatible**: `Optional[X]` from typing, `List[dict]`, `datetime.now(timezone.utc)` — no union syntax, no `datetime.UTC`
- **Generic segment builder**: `_build_segments()` in `profile_analyzer.py` handles all 5 breakdown types with one function
- **Weight normalization**: integer rounding with remainder correction ensures weights always sum to exactly 100
- **Auto-update gate**: `should_auto_update=True` only when `drift_severity=minor` AND `confidence_level=high` — conservative by default
- **Missing file handling**: feedback ingestor skips missing CSVs with a warning instead of raising; orchestrator catches `FileNotFoundError` on first run when no prior `icp_rules.json` exists
